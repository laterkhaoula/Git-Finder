
# GitHub Finder 

GitHub Finder est une application simple qui permet de rechercher des utilisateurs GitHub et d’afficher leurs informations de manière claire et rapide.

##  Comment ça marche
Tu tapes le nom d’un utilisateur GitHub, et l’application affiche :
- Sa photo de profil
- Son nom et sa bio
- Le nombre de followers et following
- Le nombre de repositories
- Un lien vers son profil GitHub

##  Fonctionnalités
- Recherche d’utilisateurs via l’API GitHub
- Affichage des profils GitHub
- Système de favoris pour sauvegarder des développeurs
- Stockage des favoris avec localStorage

##  Technologies utilisées
- HTML
- CSS
- JavaScript (Vanilla)
- GitHub API
- localStorage

##  Objectif du projet
Ce projet a été réalisé pour apprendre à travailler avec des APIs et manipuler les données en JavaScript.
